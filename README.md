# ING PSP plugin for OpenCart, v1.2.6
You can use this plugin to accept payments in your webshop with Kassa Compleet, ING ePay and ING Checkout

## Pre-requisites to install the plug-ins: 
- PHP v5.4 and above
- MySQL v5.4 and above
